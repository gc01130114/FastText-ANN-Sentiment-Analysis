# FastText Sentiment Analysis Project

## 📖 Project Overview
This project implements Sentiment Analysis using FastText word embeddings combined with an Artificial Neural Network (ANN).

## 🛠 Technologies Used
- Python
- Google Colab
- PySpark
- FastText
- TensorFlow / Keras
- Scikit-learn

## 📊 Dataset
IMDB Movie Reviews Dataset

## ⚙️ Project Workflow
1. Data Cleaning using PySpark
2. Train-Test Split (80-20)
3. Tokenization
4. FastText Pretrained Embeddings (300-d)
5. Sentence Vector Averaging
6. ANN Model Training
7. Model Evaluation (Accuracy, Precision, Recall, F1-score)

## 📈 Final Results
- Accuracy: 83.33%
- F1 Score: 0.84
